# tumblrwave
Load tumblr posts

##Requirements
* Ruby-Sass: http://sass-lang.com/install
* Gulp: http://gulpjs.com/
* Image Magick: http://www.imagemagick.org/

##Installation
Clone the repositroy and install the node dependencies 
```
npm install
```

Then you can build and run the project with gulp
```
gulp
```
